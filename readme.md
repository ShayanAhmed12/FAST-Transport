The FAST Transport Management System for FAST NUCES is a web-based solution designed to digitize and streamline university transport operations. The system introduces structured semester-based registration, automated seat allocation, route and stop management, complaint tracking, and maintenance scheduling.

Students will register for transport services each semester, select routes and stops, and receive seat assignments based on availability. If routes are full, students will be automatically placed on a waitlist. Administrators will manage buses, drivers, routes, and schedules through a centralized interface.

The system will maintain semester-wise transport fee verification records (simulated ERP integration), archive historical transport data, and provide analytical dashboards to support planning and decision-making.

Developed using Django, Django REST Framework, React.js, and PostgreSQL, the system follows a three-tier architecture and Agile Scrum methodology. The project aims to enhance transparency, reduce administrative workload, and improve overall transport service efficiency at FAST-NUCES Karachi Campus.
